¿Qué puedes hacer con nuestra plataforma?

Herramientas diseñadas para facilitar la comunicación entre ciudadanos y administradores de la ciudad.

📝

Crear reportes

Registra problemas de infraestructura o servicios públicos con título, descripción, dirección y barrio exacto.

📊

Seguimiento en tiempo real

Consulta el estado de tus reportes: Pendiente, En Progreso o Resuelto, en cualquier momento.

✏️

Editar y eliminar

Modifica o elimina tus reportes mientras estén en estado Pendiente, antes de ser procesados.

🛠️

Gestión administrativa

Los administradores analizan los reportes, cambian su estado y los remiten a la entidad encargada de dar solución.

🔐

Acceso seguro por roles

Sistema de autenticación con roles diferenciados para ciudadanos y administradores.

🗺️

Cobertura total

Todos los 187 barrios oficiales de Cartagena de Indias disponibles para localizar tu reporte.
